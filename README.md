
# VELOCITY READER ⚡️

High-performance speed reading application with gamified elements, AI-powered comprehension testing, and neural-inspired UI.

## Features
- **Speed Reading Core:** Adjustable WPM (50 - 2000), focal point guidance.
- **AI Integration:** Comprehension tests and text priming powered by Gemini 3 Flash.
- **Gamification:** Global leaderboards, XP system, and 24+ achievements.
- **Neural Squad:** Social system to track and challenge friends.
- **Strict Mode:** High-risk/High-reward training protocol with locked controls.
- **Multilingual:** Full support for EN, RU, ES, and UK.

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS
- Lucide Icons
- Google Gemini API

## Local Development
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set your API Key: `export API_KEY=your_key_here`.
4. Run development server: `npm run dev`.

## Deployment to GitHub Pages
1. Push code to `main` branch.
2. Setup GitHub Actions with the `deploy.yml` provided in the instructions.
3. Add `GEMINI_API_KEY` to repository secrets.
